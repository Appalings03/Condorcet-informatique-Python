### Commandes de base
- `print()` — affiche dans le terminal
- `del` — supprime une variable

### Opérateurs arithmétiques
- `+` — addition
- `-` — soustraction
- `*` — multiplication
- `/` — division
- `%` — modulo
- `**` — puissance
- `abs()` — valeur absolue
- `3.6e-5` — notation scientifique
- `divmod(a, b)` — quotient et reste
- `().img` — partie imaginaire d'un complexe
- `().real` — partie réelle d'un complexe

### Fonctions de conversion
- `float()` — convertit en nombre à virgule flottante
- `str()` — convertit en chaîne de caractères

### Fonctions mathématiques (NumPy)
- `np.log()`, `np.log2()`, `np.log10()`, `np.exp()`, `np.sqrt()`, `np.sin()`, `np.cos()`, `np.tan()`, `np.abs()`, `np.ceil()`, `np.floor()`, `np.round()`

### Fonctions avancées (NumPy)
- `np.linspace()` — valeurs également espacées

### Indexation et Slicing (NumPy)
- Accès aux éléments et extraction de sous-ensembles

### Fonctions de matplotlib.pyplot
